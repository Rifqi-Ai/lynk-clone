## 🚨 HIGH Severity Issues (Fix First)

### 1. Ambiguous Nama field state on Register form
- **Location:** `resources/views/auth/register.blade.php` line ~50
- **Issue:** Input has orange border (brand color) — indistinguishable from focus ring vs error
- **Heuristic violated:** Nielsen #1 (Visibility of System Status) + #9 (Help recognize errors)
- **Fix:** Use distinct error color (`#DC2626`) separate from brand orange (`#FF5722`)

### 2. Low-contrast marketing panel on Register form
- **Location:** Left gradient panel of register layout
- **Issue:** "Satu link. Tak terhingga kemungkinan." + "Bebas kartu kredit" + "Digital" badge text nearly invisible against orange gradient (~2:1 contrast)
- **Heuristic violated:** WCAG 1.4.3 Contrast (Minimum) — needs ≥4.5:1
- **Fix:** Add text-shadow OR darken gradient OR use white text with backdrop

### 3. Orange "Tak terhingga" text on cream (landing)
- **Location:** `resources/views/landing.blade.php` hero headline
- **Issue:** Borderline contrast — readable but not WCAG AA solid
- **Heuristic violated:** WCAG 1.4.3
- **Fix:** Darken brand orange or add subtle text shadow for hero text
